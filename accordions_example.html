Issuance (Branch)

BOM Checklist (Only for Mega Branches)

Guidelines & Checks

  • Customer request letter as per the format in circular forms part of scanned documents
  • Indemnity as per the format in circular forms part of scanned documents

  • Ensure margin has been collected
  • Ensure all charges have been retrieved from the customer

  • Ensure AMLOCK check done and evidenced
  • Ensure DO / SG / LOI is restricted to the LC amount
  • Ensure there is no mismatch in amount and currency
  • Validate approval from SBH has been obtained as per Annexure 68
  • Ensure details have been entered in the register and signed by Branch Operations Manager / Branch Manager (Commercial)
  • Verify that DO / SG / LOI has been signed by two officials MI and above
    - One should be either BOM / BM Commercial
